# City Research Card

Use one card per city or region. It keeps each place's facts together so you can compare places fairly later. Fill it in from trusted sources, and record each source in your Source Log.

| Prompt | Your answer |
| --- | --- |
| City or region | |
| Why people go there | |
| One memorable fact | |
| Top sight 1 (star the ones you can't wait to see) | |
| Top sight 2 | |
| Top sight 3 | |
| Food and culture notes | |
| Travel time from a likely nearby city | |
| How many days might we want here? | |
| Possible downsides (crowded? far? costly?) | |
| Best season fit (does it match our Checkpoint 1 season?) | |
| Sources (also in my Source Log) | |
| Date checked | |
| Planning assumption (something I'm assuming a grown-up should check) | |
| Needs adult verification? (yes / no) | |
| Final decision status (researching / shortlist / skip / save for future) | |

Remember: each card is a movable block. If a grown-up later changes which city you fly into or out of, you move a block -- you do not start over.
